# ¿Qué es Python?

Python es un **lenguaje de programación** que permite darle instrucciones a una computadora para ejecutar tareas de manera clara y eficiente.  

Se caracteriza por:  
- Ser **fácil de leer y escribir**.  
- Tener **sintaxis simple** y cercana al lenguaje humano.  
- Ser **muy versátil** y usado en automatización, desarrollo web, ciencia de datos, inteligencia artificial y ciberseguridad.

---

## Conceptos clave

- Python es un **lenguaje interpretado**, lo que significa que ejecuta código línea por línea sin necesidad de compilar.  
- Es un **lenguaje de alto nivel**, enfocado en la legibilidad y la facilidad de uso.  
- Se puede usar para **programación estructurada, orientada a objetos y funcional**.

---

## ¿Para qué sirve?

Python permite:  
- Automatizar tareas repetitivas  
- Manipular archivos y datos  
- Analizar información y generar gráficos  
- Crear sitios web y APIs  
- Desarrollar scripts de ciberseguridad y hacking ético  
- Implementar inteligencia artificial y machine learning  

> Nota: Cada uno de estos usos tendrá notas específicas más adelante. Por ejemplo, [[Automatización]] y [[Ciberseguridad]].

---

## Cómo funciona Python

Cuando escribes un archivo `.py`, por ejemplo:

```python
print("Hola Mundo")
````

Python realiza:

1. Lee el archivo.
2. Interpreta las instrucciones línea por línea.
3. Ejecuta el código y muestra resultados en la pantalla.

## Analogía mental

Python es como un entrenador personal: tú le das instrucciones claras y el entrenador las ejecuta.
Si la instrucción está mal escrita, devuelve un error; si está correcta, hace lo que le pediste.
